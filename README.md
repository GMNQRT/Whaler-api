# Whaler-api

The API behind [The Whaler UI](https://github.com/GMNQRT/whaler-ui).

## Getting Started

### Pre-requisites
* Computer
* Ruby 2.2.0

### Install steps
```
git clone https://github.com/GMNQRT/whaler-api.git
cd whaler-api
ruby bin/setup && rails server -b "0.0.0.0" -p 3000
```

now visit localhost:3000 and see if it works
